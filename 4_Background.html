UX Project for University of St Andrews Library (Feb 2018 - May 2018)

Outline

Develop a tool to raise awareness of other study spaces and reduce pressure on the main library

Key responsibilities
  • Primary research conducted using a qualitative questionnaire for data on the Library
  • Data distilled into 305 notes and evaluated through an affinity diagram producing 15 user requirements
  • Produced sketches to evaluate user journeys for study space finding
  • Created Lo-fidelity prototyping on whiteboards & InDesign for improvements to the tool
  • Developed a hi-fidelity prototype to show information about other spaces
Achievements
  • Project was commended and now incorporated into University of St Andrews library website
  • Received a 1st class mark for the project
